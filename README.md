# Money Muling Detection System

A Flask-based web application that detects suspicious financial transaction patterns using graph-based analysis.

## Features
- Upload transaction dataset
- Fraud / money muling detection
- Graph-based transaction analysis
- Risk scoring
- Download analysis report

## Tech Stack
- Python
- Flask
- NetworkX
- Pandas
- HTML / CSS

## Installation

```bash
pip install -r requirements.txt
python app.py
